# PickAlo Admin Dashboard

Modern web-based admin dashboard for the PickAlo sports venue booking platform.

## Features

- Dashboard with platform metrics
- User management (ban/unban, role changes)
- Venue verification workflow
- Booking oversight (admin cancel)
- Content moderation (posts/comments)
- Audit log viewer

## Tech Stack

- React 19 + TypeScript
- Vite 6
- TanStack Query v5
- React Router v7
- TailwindCSS

## Getting Started

```bash
npm install
npm run dev
```

Dashboard will be available at http://localhost:3000

## Backend

Connects to FastAPI backend at http://localhost:8000
